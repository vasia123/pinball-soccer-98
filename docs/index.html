Soccer Pinball ’98

Легендарный футбольный пинбол эпохи Windows 98 — снова живой, прямо в браузере. Без установки, без дискет, без ностальгической тоски.

▶ Работает целиком у вас в браузере · ничего не отправляется на сервер
DYNAMITE SOCCER PINBALL — TOURNAMENT TABLE
▸ ИГРАТЬ
Первая загрузка занимает 20–40 секунд: в браузере поднимается Windows-совместимая среда и распаковывается стол.

Управление

◄ Shift левый флиппер
Shift ► правый флиппер
Space запуск мяча (плунжер)
Enter подтвердить в меню
Esc назад / выход из стола

Как начать

  • Нажмите ИГРАТЬ и дождитесь загрузки.
  • Выберите язык (флаг) и нажмите GO.
  • В меню стола выберите разрешение/цвет и снова GO.
  • Ловите мяч флипперами. Кнопки в меню нажимайте с чуть удержанным кликом.
